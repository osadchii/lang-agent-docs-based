# Телеграм-бот для изучения языков

## Цели и фокус
- Предназначен для русскоязычных пользователей Telegram, изучающих иностранные языки (LLM позволяет изучать любой язык: английский, немецкий, французский и др.).
- Предоставляет диалоговый LLM-помощник: объясняет грамматику и словообразование, переводит слова и предложения, помогает разобрать ошибки.
- Поддерживает активную практику: флешкарточки с упором на грамматические категории (падежи, времена, роды), интерактивные задания и тесты по запросу пользователя.
- Обеспечивает максимально простое добавление материала: пользователь может создавать карточки и упражнения «на лету» прямо в диалоге — выделить слово в сообщении, попросить бота добавить его, сфотографировать текст или отправить голосовое, без необходимости целенаправленно загружать списки слов.
- Работает с изображениями: распознаёт текст на фото/сканах, переводит, проверяет или выполняет упражнения.
- Состоит из Python-бэкенда (Telegram-бот + API) и React Telegram Mini App; функциональность бота дублируется, но в мини-приложении фокус на удобной работе с карточками и заданиями.
- Включает социальные механики: группы обучения, совместное использование колод, заданий и прогресса; поддерживает роли «учитель» и «ученик».
- Монетизируется через подписку, предусматривает ручное включение премиума администратором.
- Использует стек Python + React + PostgreSQL; деплой автоматизирован GitHub Actions с последующим развёртыванием на сервере.
